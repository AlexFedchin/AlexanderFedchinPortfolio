## Getting Started

```sh
# Clone the repo
git clone https://github.com/AlexFedchin/AlexanderFedchinPortfolio.git
cd AlexanderFedchinPortfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Other scripts:

```sh
npm run build      # Production build
npm run preview    # Preview production build locally
npm run lint       # Run ESLint
npm run test       # Run Vitest tests
```
